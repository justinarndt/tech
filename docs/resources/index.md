## Featured Resources

### For Beginners

Start here if you're new to technical writing:

1. **Google Technical Writing Courses** (Free)
   Excellent foundation in technical writing principles.

2. **Write the Docs Community**
   Welcoming community with active discussions.

3. **Google Developer Documentation Style Guide**
   Industry-standard reference for technical content.

### For Growing Skills

Expand your capabilities:

1. **Docs for Developers** (Book)
   Comprehensive guide to developer documentation.

2. **LinkedIn Learning** (Subscription)
   Wide range of technical writing courses.

3. **API Documentation Courses**
   Specialize in high-demand skills.

### For Leadership

Advance to the next level:

1. **Content Strategy Resources**
   Think strategically about documentation.

2. **Management Resources**
   Lead teams effectively.

3. **Industry Conferences**
   Learn from leaders and expand network.

## How to Use These Resources

### Learning Approach

```markdown
## Recommended Learning Path

### Phase 1: Foundations
- Complete Google Tech Writing courses
- Read one foundational book
- Join Write the Docs community

### Phase 2: Building Skills
- Study relevant style guide
- Take tool-specific courses
- Practice with real projects

### Phase 3: Specialization
- Deep dive into chosen area
- Advanced courses and books
- Industry involvement
```

### Staying Current

- Subscribe to 2-3 newsletters
- Follow industry blogs
- Participate in communities
- Attend events when possible
