# Screenshot and Recording

Screenshots and recordings show users exactly what to expect. The right tools make capturing and annotating visual content efficient.

## Screenshot Tools

### Built-in Options

**macOS:**
- Cmd+Shift+4: Selection screenshot
- Cmd+Shift+5: Screenshot toolbar with options

**Windows:**
- Win+Shift+S: Snipping tool
- Print Screen: Full screen capture

### Professional Tools

**Snagit:**
- Annotation features
- Scrolling capture
- Video recording
- Organized library

**CleanShot X (macOS):**
- Clean backgrounds
- Annotations
- Cloud storage
- GIF recording

## Screen Recording

### Tools

- **OBS Studio**: Free, powerful, cross-platform
- **Loom**: Quick sharing, cloud-based
- **Camtasia**: Full editing suite
- **QuickTime (macOS)**: Simple, built-in

### Best Practices

1. **Plan before recording**: Know what you'll show
2. **Clean your desktop**: Hide personal items
3. **Use a script**: Avoid rambling
4. **Record at consistent size**: Usually 1920x1080
5. **Keep videos short**: 2-5 minutes ideally

## Image Best Practices

### Screenshot Guidelines

- Capture only what's needed
- Use consistent window sizes
- Add annotations sparingly
- Optimize file size

### Accessibility

- Include alt text
- Don't rely on images alone
- Ensure text is readable

## Summary

Visual content enhances documentation when used well:

- Use appropriate tools for your platform
- Maintain consistency in captures
- Keep visuals focused and clear
- Always consider accessibility
